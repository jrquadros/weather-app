## Technologies
* Typescript
* React Native
* Styled Components
* Redux


### Running the project
_Requires `react-native-cli` installed_

1. Clone the project
2. Install dependencies with `yarn`
3. Start the dev server with `yarn start`
4. Run with `react-native run-android`

Linter: `yarn lint`
